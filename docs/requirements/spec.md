# Game Description
I want a game involving unicorn kitties. They should be able to swim in a pond and catch fish. They should also be able cook bacon on the shoreline on a grill. They should be able to build houses as well.

## Technical Details
This game should be as simple as possible and playable in a browser.  It should use sleek, modern UI standards.  The player should be able to move the unicorn kitties around with the arrow keys (left, down, right, up) and jump with the space bar.  They should be able to enter houses with the 'return' or 'enter' keys. They should get points for every fish they catch or bacon they successfully cook.

## UI Details
There should be rainbows in the sky at night. The overall theme is kid-friendly with characters with big eyes and friendly expressions.  There should be various background characters around.  The fish should be fun looking.

## Future work
There should be a character create screen where you can design your own character using LLM integration.
